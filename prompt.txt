# Prompt

請用 vibe coding 的方式幫我寫一個 JavaScript 小遊戲，要求如下：

1. 遊戲名稱為 **Avoid Game (躲避遊戲)**。
2. 玩家可以左右移動藍色方塊，避開紅色方塊。
3. 遊戲分數會隨時間增加。
4. 增加不同顏色方塊的效果：
   - 紅色 (Red)：碰到即死亡 (Game Over)。
   - 綠色 (Green)：碰到可額外加 10 分。
   - 黃色 (Yellow)：碰到會扣 5 分。
5. 畫面設計要有 **像素復古風格**：
   - 使用像素字體。
   - 背景有花紋（例如像素斜紋）。
   - 按鈕設計成復古街機風格。
6. 遊戲介面使用 **英文**。
7. 遊戲結束後會顯示分數，並保存最高分紀錄。
8. 技術：HTML / CSS / JavaScript (純前端)，可部署於 GitHub Pages。
